## 📜 Descrição

O Brasil vive um cenário preocupante ao se tratar do assunto de saúde financeira, com grande parte dos jovens sem controle de gastos e enfrentando inadimplência. Para enfrentar esse problema, foi desenvolvido o jogo Mestre dos Sistemas, que coloca o jogador no papel de um consultor bancário iniciante que, dia após dia, atende clientes com situações financeiras que simulam a realidade envolvendo crédito, débito, pré-pago, empréstimos, juros e planejamento financeiro. Por meio de uma simulação em pixel art no formato point-and-click, o jogador é exposto a decisões com consequências diretas dentro da narrativa, promovendo o desenvolvimento do raciocínio financeiro de forma aplicada e progressiva. Desenvolvido em parceria com a Mastercard e disponível para web e mobile, o jogo se apresenta como uma ferramenta de educação financeira voltada ao público jovem brasileiro.

*Link para o jogo: https://graduacao.pages.git.inteli.edu.br/2026-1a/t24/g05/*

## 📁 Estrutura de pastas

Dentre os arquivos e pastas presentes na raiz do projeto, definem-se:

- <b>assets</b>: aqui estão os arquivos relacionados a elementos não-estruturados deste repositório, como imagens.

- <b>document</b>: aqui estão todos os documentos do projeto, como o Game Development Document (GDD) bem como documentos complementares, na pasta "other".

- <b>src</b>: Todo o código fonte criado para o desenvolvimento do projeto do jogo.

- <b>.gitlab</b>: Salva os templates usados dentro do gitlab para a criação dos cards no "issue boards".

- <b>README.md</b>: arquivo que serve como guia e explicação geral sobre o projeto e o jogo (o mesmo que você está lendo agora).

- <b>.gitlab-ci.yml</b>: arquivo que configura o deploy automático do projeto no GitLab Pages, ou seja, sempre que um commit é feito na branch principal, ele publica automaticamente a pasta public/ como um site acessível online.

```
├───.gitlab
|       ├───issue_templates
├───assets
|       ├───clientes
|       ├───concepts
|       ├───conquistas
|       ├───fluxograma
|       ├───negócios
|       ├───screenshots
|       ├───sprites
|
| 
├───documents
|       ├───gdd.md
|
└───public
        ├───.vscode
        ├───src
        ├───index.html
        ├───phaser.js
```
## 🔧 Como executar o código

### **Pré-Requisitos:**

**Visual Studio Code**;
**Navegador WEB**. como Google Chrome, Microsoft Edge, ou Mozilla Firefox, para visualização e teste do jogo.
**Extensão Live Server do VSCode**

Após a garantia de todos os pré-requisitos, instale o projeto assim:

- Clone e faça o download desse repositório:
```
https://git.inteli.edu.br/graduacao/2026-1a/t24/g05.git
```

### Instruções de execução

  **1. Abra o Visual Studio Code**
  
  **2. Dentro do Visual Studio Code, clique em "Open Folder..."**

  **3. Selecione a pasta do repositório que acabou de baixar**

  **4. Clique com o botão direito em "index.html" e selecione "Open with Live Server"**
